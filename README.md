# sales-Dashboard

📌 Project Overview
This project presents a Sales Analysis Dashboard created using Microsoft Power BI.
The dashboard helps visualize and analyze sales performance based on different factors such as region, category, segment, and order date.
The goal of this project is to transform raw sales data into interactive visual insights for better business decision-making.

📂 Dataset Information
The dataset contains the following columns:
Order Date
Region
Category
Segment
Ship Mode
Sales
These fields are used to create different visualizations and KPIs in the dashboard.

📊 Dashboard Features
The Power BI dashboard includes the following visualizations:
Total Sales KPI Card – Shows overall sales value
Sales by Region (Donut Chart) – Compares sales across regions
Sales by Segment (Pie Chart) – Displays customer segment contribution
Sales by Category (Bar Chart) – Shows performance of product categories
Sales Trend by Order Date (Line Chart) – Analyzes sales growth over time
Sales by Region (Column Chart) – Region-wise comparison

🎯 Key Insights
Identify top-performing regions
Understand customer segment contribution
Track sales trends over time
Compare product category performance
